# Act0 color palette + modified tiles for Diablo 2

This repository provides the **Act0 color palette** and modified tilesets for **Diablo 2**, designed to work across all acts of the game without color shifting issues or graphical bugs.

## [Graphics preview (beta)](https://d2r-gimli.github.io/Diablo2_act0_color_palette/)


## Overview

- The original Diablo 2 game uses different **256-color palettes** for each act.
- The **Act0 color palette** included here contains only colors available in **all acts**, making it act-independent.
- While this reduces the total number of colors, any graphics saved with this palette will **work consistently across all acts**.

## Usage

- Use the Act0 palette for **consistent graphics across all acts**.
- Tilesets can be imported directly into map editors or projects.
- For tiles marked with specific act palettes, follow the instructions in the empty marker file.
